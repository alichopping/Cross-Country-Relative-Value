We present a systematic cross-sectional rates trading strategy for 10-year government bond futures across a set of six major developed markets. The approach seeks to exploit relative mispricings in short-horizon, risk-adjusted returns using economically motivated features, including momentum and yield curve structure. We construct a panel of volatility-scaled forward-looking returns and perform a rolling ridge regression to forecast relative performance across a universe of six government bond futures (US, UK, Germany, Canada, France, and Italy). The forecast is then used to construct a relative value portfolio. The strategy demonstrates some degree of predictive power as measured by the information coefficient, though the signal is noisy for reasons we discuss. The strategy shown to be orthogonal to basic momentum, and the residual returns above momentum are positive before transaction costs. However, we find that the signal is highly sensitive to trading frictions - transaction costs fully erode the residual returns in a naive execution strategy. We discuss key limitations of our work, including the small cross-sectional universe, the absence of explicit futures roll adjustment, and the need for a more sophisticated execution strategy.
